Sistema web para gestionar cursos y matrículas universitarias.

## Tecnologías
- ASP.NET Core MVC (.NET 9)
- Entity Framework Core + SQLite
- ASP.NET Identity
- Redis (sesiones y caché)
- Bootstrap 5
- Deploy en Render.com
